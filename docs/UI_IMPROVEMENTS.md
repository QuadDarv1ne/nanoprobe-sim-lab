### 2. Новые CSS классы

#### `.stats-grid.compact`
- Уменьшенные отступы (0.5rem вместо 1rem)
- Минимальная ширина карточки 100px вместо 140px
- Плотная компоновка

#### `.stat-card.compact`
- Уменьшённый padding (0.625rem вместо 1.25rem)
- Меньший размер шрифта (1.375rem вместо 2rem)
- Анимация при наведении

#### `.stat-badge`
- Новый тип мини-статистики
- Горизонтальное расположение
- Идеально для системных метрик

---

### 3. Цветовая индикация

**CPU/RAM/Disk:**
- 🟢 **0-50%**: зелёный/синий (норма)
- 🟡 **50-80%**: жёлтый (внимание)
- 🔴 **80-100%**: красный (критично)

**Компоненты:**
- ⚫ **0**: серый (нет активных)
- 🟢 **1-2**: зелёный (норма)
- 🔵 **3+**: синий (много активных)

---

### 4. Улучшенный uptime

**Формат:**
- `< 1 часа`: `"15м"`, `"45м"`
- `1-24 часа`: `"1ч 30м"`, `"12ч"`
- `> 24 часов`: `"24ч 15м"`, `"48ч"`

## 🔧 Использование

### Основная статистика
```html
<div class="stats-grid compact">
    <div class="stat-card compact">
        <div class="stat-value">123</div>
        <div class="stat-label">Сканирований</div>
    </div>
</div>
```

### Системные метрики
```html
<div class="stats-row">
    <div class="stat-badge">
        <div class="stat-badge-value">25%</div>
        <div class="stat-badge-label">CPU</div>
    </div>
</div>
```

### JavaScript обновление
```javascript
// Обновление значения
document.getElementById('stat-cpu').textContent = '25%';

// С цветовой индикацией
const cpuEl = document.getElementById('stat-cpu');
cpuEl.textContent = cpuPercent.toFixed(1) + '%';
if (cpuPercent > 80) {
    cpuEl.style.color = 'var(--danger)';
} else if (cpuPercent > 50) {
    cpuEl.style.color = 'var(--warning)';
} else {
    cpuEl.style.color = 'var(--primary)';
}
```
