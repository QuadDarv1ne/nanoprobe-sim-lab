# Nanoprobe Sim Lab - Проект Заметки

## 📦 Контекст Проекта

**Название:** Nanoprobe Sim Lab  
**Описание:** Лаборатория моделирования нанозонда с SSTV ground station  
**Основное оборудование:** RTL-SDR V4 (ожидается через 2 недели)  
**Python:** 3.13+  
**OS:** Windows 11  

---

## 🎯 Ключевые Функции

### SSTV Ground Station
- Real-time декодирование SSTV с МКС
- Поддержка режимов: Martin, Scottie, PD, Robot
- Автодетект сигнала
- Сохранение изображений с timestamp

### Спутниковое Отслеживание
- TLE данные для ISS, NOAA 15/18/19, Meteor-M2
- Предсказание пролётов на 24 часа
- Частоты: ISS=145.800, NOAA15=137.620, NOAA18=137.9125, NOAA19=137.100

### Waterfall Дисплей
- FFT 512 bins в реальном времени
- Сохранение PNG/GIF
- Частота обновления ~10 fps

## 🔧 Конфигурация

### RTL-SDR V4 Настройки
```python
sample_rate = 2400000  # 2.4 MSPS
gain = 30  # dB (0-50)
bias_tee = True  # для активной антенны
agc = True  # авто регулировка усиления
```

### Наземная Станция (по умолчанию)
```python
latitude = 55.75   # Москва
longitude = 37.61
```

## 🚀 Основные Команды

```bash
# Проверка устройства
python components/py-sstv-groundstation/src/main.py --check

# Real-time SSTV с МКС
python main.py --realtime-sstv -f iss --duration 120 --gain 35

# Waterfall спектр
python main.py --waterfall -f 145.800 --save-waterfall

# Расписание спутников
python main.py --schedule --lat 55.75 --lon 37.61

# Веб-интерфейс
python src/web/web_dashboard.py
```
