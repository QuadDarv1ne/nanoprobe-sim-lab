# Лаборатория моделирования нанозонда (Nanoprobe Simulation Lab)

## Описание проекта

**Лаборатория моделирования нанозонда** - это комплексный научно-образовательный проект, объединяющий три взаимосвязанных модуля для моделирования и анализа нанотехнологий:

1. **Симулятор аппаратного обеспечения СЗМ** (C++/Python) - моделирование сканирующей зондовой микроскопии
2. **Анализатор изображений поверхности** (Python) - обработка и анализ AFM-изображений
3. **Наземная станция SSTV** (Python/C++) - прием и декодирование радиосигналов
4. **FastAPI REST API** ⭐ - Полноценный REST API с JWT и WebSocket
