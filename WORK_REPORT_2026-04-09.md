## 📝 Технические детали

### pysstv структура модулей (v0.5.7)

```
pysstv/
├── sstv.py          # Базовый класс SSTV
├── color.py         # Цветные режимы (PD120, Martin, Scottie, Robot36)
├── grayscale.py     # Ч/Б режимы (Robot8BW, Robot24BW)
└── ...
```

**Важно:** В версии 0.5.7 режимы перемещены из `pysstv.mode` в `pysstv.color`

### API эндпоинты SSTV

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/v1/sstv/recordings` | Список записей |
| GET | `/api/v1/sstv/recordings/{filename}` | Скачать запись |
| DELETE | `/api/v1/sstv/recordings/{filename}` | Удалить запись |
| POST | `/api/v1/sstv/record/start` | Начать запись |
| POST | `/api/v1/sstv/record/stop` | Остановить запись |
| GET | `/api/v1/sstv/record/status` | Статус записи |
