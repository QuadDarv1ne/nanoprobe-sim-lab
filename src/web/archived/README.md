# Archived Flask Dashboards

## Файлы в этой папке

- `web_dashboard.py` — Оригинальный Flask дашборд (~1669 строк)
- `web_dashboard_integrated.py` — Интегрированная версия с reverse proxy (~1270 строк)

## Почему заархивированы

Эти файлы были эволюционными итерациями веб-дашборда. Они дублировали ~70-80% кода друг друга.

**Актуальный файл:** `web_dashboard_unified.py` (компактная версия, ~715 строк)

## Альтернатива

Проект мигрирует на **Next.js frontend** (`frontend/` директория) как основной UI.

### Когда использовать эти файлы

- Только для исторической справки или отката
- Не использовать для новой разработки
- Могут быть удалены после полной миграции на Next.js
