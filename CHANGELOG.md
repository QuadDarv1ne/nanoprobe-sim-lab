# История изменений Nanoprobe Sim Lab

## [1.0.0] - 2026-03-11

### ✨ Новые возможности

#### FastAPI REST API
- Добавлен полноценный REST API на FastAPI
- JWT аутентификация с access/refresh токенами
- CRUD операции для сканирований и симуляций
- AI/ML анализ дефектов через API
- Сравнение поверхностей с метриками (SSIM, PSNR, MSE)
- Генерация PDF отчётов через API
- WebSocket для real-time обновлений
- Автодокументация (Swagger UI, ReDoc)

#### Администрирование
- Панель администратора для управления системой
- Мониторинг ресурсов (CPU, память, диск)
- Управление пользователями
- Просмотр и очистка логов
- Оптимизация базы данных (VACUUM, ANALYZE)
- Управление кэшем

#### CLI утилиты
- `admin_cli.py` - утилита командной строки для администратора
- Команды: status, cleanup, backup, restore, users, migrate, info
- `deploy.sh` - скрипт автоматического развёртывания
- `monitor.sh` - скрипт мониторинга системы

#### Документация
- `docs/API.md` - полная документация API
- `docs/ADMIN.md` - руководство администратора
- `docs/DEVELOPER.md` - руководство разработчика
- `api/README.md` - быстрый старт API
- `INSTALL.md` - инструкция по установке
- `CHANGELOG.md` - история изменений

## Формат версий

Проект использует [Semantic Versioning](https://semver.org/lang/ru/):

- **MAJOR** - обратно несовместимые изменения
- **MINOR** - новые возможности (обратно совместимые)
- **PATCH** - исправления ошибок

## Типы изменений

- **✨ Added** - новые возможности
- **🔧 Changed** - изменения в существующем
- **🐛 Fixed** - исправления ошибок
- **⚡ Performance** - улучшения производительности
- **📝 Documentation** - документация
- **🧪 Tests** - тесты
- **🔒 Security** - безопасность
