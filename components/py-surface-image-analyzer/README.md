# Анализатор изображений поверхности на Python

Приложение на Python для обработки и анализа изображений, имитирующих сканирование микроскопом. Этот инструмент предоставляет функциональность для фильтрации, анализа и визуализации характеристик поверхности из изображений в формате AFM/SPM.

## Цель

Разработать программу для обработки и анализа изображений, имитирующих сканирование микроскопом, с возможностями фильтрации шума, измерения свойств поверхности и визуализации результатов.

## ✨ Особенности

- **Обработка изображений**: Использование OpenCV/scikit-image для загрузки сканирований (начиная с текстур, позже реальные сканирования AFM/SPM из открытых баз данных)
- **Фильтрация**: Удаление шума (космические лучи, помехи) с помощью медианных или гауссовых фильтров
- **Анализ**: Расчет шероховатости поверхности (среднеквадратичное отклонение высот), обнаружение и измерение дефектов (пор или кратеров) по контрасту
- **Визуализация**: Построение 3D-поверхностей через matplotlib, выделение обнаруженных дефектов цветом

## 🐍 Почему Python

Python идеален для этого приложения потому что:
- Богатая экосистема научных библиотек (NumPy, SciPy, OpenCV)
- Простота прототипирования и экспериментирования
- Сильные возможности визуализации
- Возможности интеграции с машинным обучением

## 🚀 Начало работы

### Необходимые условия
- Python 3.8 или выше
- менеджер пакетов pip

### Установка
```bash
pip install -r requirements.txt
```

Или установите зависимости вручную:
```bash
pip install numpy matplotlib opencv-python scikit-image scikit-learn
```

### Запуск
```bash
python main.py --input sample_image.jpg --output analyzed_result.png
```

## 🧩 Модули

### Обработчик изображений
Обрабатывает загрузку, предварительную обработку и базовую манипуляцию с изображениями поверхности.

### Анализатор поверхности
Реализует алгоритмы для вычисления свойств поверхности, таких как шероховатость, обнаружения дефектов и измерения характеристик.

### Фильтр шума
Предоставляет различные методы фильтрации для очистки изображений и снижения уровня шума перед анализом.

### Визуализатор
Создает 3D-представления поверхности и выделяет обнаруженные особенности.
