# Симулятор аппаратного обеспечения СЗМ на C++

Консольное приложение на C++, которое моделирует работу сканирующего зондового микроскопа (СЗМ), включая сканирование поверхности по сетке и сбор моделируемых топографических данных.

## Цель

Создать консольное приложение, которое моделирует работу сканирующего зондового микроскопа (СЗМ), включая сканирование поверхности по сетке и сбор моделируемых топографических данных.

## ✨ Особенности

- **Модель поверхности**: Загрузка из файла или генерация матриц высот с заданной шероховатостью, трещинами или кратерами
- **Модель зонда**: Реализация алгоритмов движения по сетке X,Y с регулировкой высоты Z для поддержания контакта с поверхностью
- **Сбор данных**: Моделирование механизмов обратной связи (например, увеличение значения Z при сканировании "гор")
- **Визуализация**: Вывод результатов в консоль в виде ASCII-арта или генерация файлов для 2D/3D построения графиков

## 🛠️ Почему C++

C++ позволяет практиковаться с массивами данных, точной математикой и логикой, близкой к реальному программному обеспечению управления оборудованием. Он идеален для:
- Операций, чувствительных к производительности
- Низкоуровневого управления моделируемым оборудованием
- Эффективных матричных и векторных вычислений

## 🚀 Начало работы

### Необходимые условия
- Компилятор C++ (рекомендуется C++17 или выше)
- CMake (необязательно, для сборки)

### Сборка
```bash
mkdir build
cd build
cmake ..
make
```

### Запуск
```bash
./spm-simulator
```

## 🧩 Модули

### Модель поверхности
Обрабатывает генерацию и загрузку данных о топографии поверхности. Поддерживает как процедурную генерацию, так и загрузку на основе файлов.

### Модель зонда
Моделирует физическое движение зонда и взаимодействие с поверхностью, включая механизмы обратной связи.

### Контроллер СЗМ
Управляет общим процессом сканирования, координирует движение зонда и собирает данные.
