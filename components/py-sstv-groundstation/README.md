# 📡 SSTV Ground Station - Приём SSTV с МКС и спутников

Полнофункциональная наземная станция для приёма и декодирования SSTV сигналов с Международной Космической Станции (МКС) и других спутников.

## 🚀 Возможности

- ✅ **Приём SSTV** с МКС на частоте 145.800 MHz
- ✅ **RTL-SDR V4 поддержка** - полная совместимость с R828D тюнером
- ✅ **Автоматическое декодирование** - поддержка режимов Martin, Scottie, PD, Robot, Wraase
- ✅ **Отслеживание спутников** - реальные TLE данные из CelesTrak + SGP4 propagation
- ✅ **Waterfall спектрограмма** - визуализация спектра в реальном времени
- ✅ **Автоматическая запись** - запись при пролёте спутника
- ✅ **Расписание пролётов** - предсказание видимости спутников
- ✅ **CLI интерфейс** - удобные команды для всех функций

## 🛰 Как работает отслеживание спутников

### Реальные TLE данные

SatelliteTracker загружает **актуальные Two-Line Element (TLE)** данные напрямую с **CelesTrak** - официального источника орбитальных данных.

### SGP4 Propagation

Используется библиотека `sgp4` для точного вычисления позиции спутника на основе TLE:

1. **Загрузка TLE** из CelesTrak при каждом запуске
2. **SGP4 вычисление** позиции в реальном времени
3. **Расчёт elevation** угла для определения видимости
4. **Предсказание пролётов** с точностью до 30 секунд

### Точность

- **Позиция**: ±1-2 км (зависит от возраста TLE)
- **Время пролёта**: ±1-2 минуты
- **Elevation**: ±5-10 градусов

## 🎓 SSTV - Что это?

**SSTV (Slow Scan Television)** - это метод передачи изображений по радио. Используется на МКС и спутниках для передачи фотографий и экспериментов.

### Как принимать SSTV с МКС:

1. **Проверьте расписание** пролётов МКС
2. **Подключите антенну** к RTL-SDR (лучше всего четвертьволновая для 2m)
3. **Запустите запись** за 5 минут до пролёта
4. **Декодируйте** запись после пролёта

### Советы по приёму:

- 📡 Используйте внешнюю антенну (четвертьволновая для 145.8 MHz ≈ 51 см)
- 🏔️ Разместите антенну на открытом месте
- 📍 Проверьте что МКС действительно виден в вашем районе
- ⏱ Начните запись за 2-3 минуты до пролёта
- 🎚 Используйте gain 40-50 dB для лучшего приёма

## 🐛 Решение проблем

### "RTL-SDR устройства не обнаружены"

1. Проверьте подключение USB
2. Установите драйверы Zadig (WinUSB для RTL-SDR)
3. Запустите `setup_rtlsdr.bat`
4. Перезагрузите компьютер

### "Не удалось декодировать изображение"

- МКС может не передавать SSTV в данный момент
- Проверьте расписание пролётов
- Убедитесь что антенна подключена
- Попробуйте увеличить gain

### "TLE не загружаются"

- Проверьте интернет-соединение
- TLE кэшируются локально в `data/tle_data.json`
- Можно использовать сохранённые TLE без интернета
